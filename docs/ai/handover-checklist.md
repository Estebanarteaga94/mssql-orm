# Checklist de handoff

Antes de cerrar una sesión, verificar:

- [ ] La tarea activa fue movida correctamente en `docs/tasks.md`
- [ ] `docs/worklog.md` registra fecha, cambios, resultado, bloqueos y siguiente paso
- [ ] `docs/context.md` refleja el estado operativo vigente
- [ ] Las validaciones ejecutadas quedaron claras
- [ ] Los límites arquitectónicos no fueron violados
- [ ] No se incluyeron archivos ajenos en el commit
- [ ] El siguiente paso recomendado quedó explícito
